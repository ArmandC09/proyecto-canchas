Pasos rápidos:
1. Importa database/alquilatucancha_db.sql en MySQL.
2. Edita php/conexion.php si tu usuario, contraseña o nombre de base cambian.
3. Abre frontend/registro-usuario.php o frontend/registro-propietario.php para crear cuentas.
4. Luego inicia sesión desde frontend/login.php.
